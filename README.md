# Juego Hundir la Flota - Proyecto PRG
Este es un proyecto de programación estructurada en Python para la asignatura de Programación.

## Estructura del Proyecto
- `main.py`: Controlador principal del juego.
- `tablero.py`: Gestión de las matrices y visualización.
- `barcos.py`: Lógica de posicionamiento de la flota.
- `mecanicas.py`: Reglas de disparo e IA.
- `interfaz.py`: Interacción con el usuario.

## Cómo ejecutar
Ejecutar el comando: `python main.py`
